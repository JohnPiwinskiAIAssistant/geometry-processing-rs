# Projects

This module contains implementations of various geometry processing algorithms. Each project demonstrates a specific technique or application.

## Parameterization

### [Parameterization](./parameterization.md)
Algorithms for flattening 3D surfaces to 2D, including:
- Spectral Conformal Parameterization
- Boundary First Flattening

## Scalar Field Processing

### [Poisson Problem](./poisson_problem.md)
Solving Poisson equations on surfaces for scalar field reconstruction.

### [Geodesic Distance](./geodesic_distance.md)
Computing geodesic distances using the heat method.

### [Harmonic Bases](./harmonic_bases.md)
Computing harmonic basis functions for spectral analysis.

## Curvature and Flow

### [Discrete Curvatures](./discrete_curvatures.md)
Computing various discrete curvature measures.

### [Geometric Flow](./geometric_flow.md)
Mean curvature flow and other geometric flows.

## Vector Fields

### [Direction Field Design](./direction_field_design.md)
Designing smooth direction fields on surfaces.

### [Vector Field Decomposition](./vector_field_decomposition.md)
Helmholtz-Hodge decomposition of vector fields.

## Topology

### [Tree-Cotree](./tree_cotree.md)
Tree-cotree decomposition for computing homology generators.

### [Simplicial Complex Operators](./simplicial_complex_operators.md)
Building matrices for discrete exterior calculus.

### [DEC Project](./discrete_exterior_calculus_project.md)
Discrete exterior calculus operators and applications.
