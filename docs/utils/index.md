# Utils

This module contains utility functions and helpers for common geometry processing tasks.

## [Solvers](./solvers.md)
Numerical solvers for eigenvalue problems and linear systems.

## [Mesh I/O](./meshio.md)
Reading and writing mesh files in various formats (OBJ, etc.).

## [Colormap](./colormap.md)
Color mapping utilities for visualization.

## [Distortion](./distortion.md)
Computing and visualizing geometric distortion metrics.
